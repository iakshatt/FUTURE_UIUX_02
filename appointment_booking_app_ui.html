UX flow

5-screen linear flow: Onboarding → Login → Services → Date/Time → Confirmation. Progressive disclosure keeps users from being overwhelmed — each screen has a single primary action.

Visual hierarchy

Purple brand color (#534AB7) anchors trust. Category color-coded specialist cards let users scan and compare instantly. Booked slots are dimmed — not hidden — to prevent confusion.

Mobile-first choices

44px minimum tap targets on all interactive elements. Sticky bottom nav for one-thumb reachability. Notes field is optional to reduce friction. Social login reduces sign-in drop-off.

Real-world UX

Confirmation includes Booking ID, calendar add, share, and directions — the 4 things users need post-booking. Email confirmation sets expectations and reduces no-shows.
